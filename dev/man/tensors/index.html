Tensors

A tensor, parametrized by an order and a dimension, is in general defined by

  • an array or a set of condensed parameters (e.g. isotropic tensors),
  • a basis,
  • a set of variances (covariant :cov or contravariant :cont) useful if the basis is not orthonormal.

In practice, the type of basis conditions the type of tensor (TensCanonical, TensRotated, TensOrthogonal, Tens or even TensISO in case of isotropic tensor).

Special tensors are available

  • tensId2(::Val{dim} = Val(3), ::Val{T} = Val(Sym)) where {dim,T<:Number}: second-order identity (рқҹҸбөўвұј = Оҙбөўвұј = 1 if i=j otherwise 0)
  • tensId4(::Val{dim} = Val(3), ::Val{T} = Val(Sym)) where {dim,T<:Number}: fourth-order identity with minor symmetries (рқ•Җ = рқҹҸ вҠ Лў рқҹҸ i.e. (рқ•Җ)бөўвұјвӮ–вӮ— = (ОҙбөўвӮ–ОҙвұјвӮ—+ОҙбөўвӮ—ОҙвұјвӮ–)/2)
  • tensJ4(::Val{dim} = Val(3), ::Val{T} = Val(Sym)) where {dim,T<:Number}: fourth-order spherical projector (рқ•Ғ = (рқҹҸ вҠ— рқҹҸ) / dim i.e. (рқ•Ғ)бөўвұјвӮ–вӮ— = ОҙбөўвұјОҙвӮ–вӮ—/dim)
  • tensK4(::Val{dim} = Val(3), ::Val{T} = Val(Sym)) where {dim,T<:Number}: fourth-order deviatoric projector (рқ•Ӯ = рқ•Җ - рқ•Ғ i.e. (рқ•Ӯ)бөўвұјвӮ–вӮ— = (ОҙбөўвӮ–ОҙвұјвӮ—+ОҙбөўвӮ—ОҙвұјвӮ–)/2 - ОҙбөўвұјОҙвӮ–вӮ—/dim)
  • ISO(::Val{dim} = Val(3), ::Val{T} = Val(Sym)) where {dim,T<:Number}: returns рқ•Җ, рқ•Ғ, рқ•Ӯ

The useful tensor products are the following:

  • вҠ— tensor product
  • вҠ—Лў symmetrized tensor product
  • вҠ  modified tensor product
  • вҠ Лў symmetrized modified tensor product
  • вӢ… contracted product
  • вҠЎ double contracted product
  • вҠҷ quadruple contracted product

NOTE: more information about modified tensor products can be found in SГ©bastien Brisard's blog.
